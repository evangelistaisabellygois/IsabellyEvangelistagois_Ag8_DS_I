#Sistema de avaliação

O projeto foi desenvolvido totalmente em python com o objetivo de ajudar a melhorar o atendimento e feedback.
apresenta três opções de feedback:

-Perfeito 😃
-Bom 🙂
-Ruim ☹️

Utilizando (`if` `elif` e `elf`) para identificar a opção escolhida e mostrar a mensagem correspondente.

#🕹 Tecnologia utilizada: python

#📖 Conhecimentos aplicados 
- `print`
- `if`
- `elif`
- `elf`
- variáveis
- saída de dados
